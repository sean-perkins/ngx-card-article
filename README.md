# ngx-card-article

Beautifully designed cards for use in an Angular 4 project.

1. `npm i ngx-card-article --s`

![Example Image](http://i.imgur.com/MGG1dMF.jpg)


## Usage

```
<ngx-card-article
        [title]="'My Title'"
        [photoUrl]="'http://www.example.com/image.jpg'"
        [startColor]="'#fff'"
        [endColor]="'rgba(50, 50, 50, 0.1)'">
        <div header>
            <li>Testing</li>
            <li>Testing</li>
        </div>
        <div footer>
            <li>Testing</li>
            <li>Testing</li>
        </div>
</ngx-card-article>
```
